## вЬ® Core Features

### рЯОѓ Interactive Kanban Workspace
- **Native HTML5 drag-and-drop** вАФ no third-party DnD library, zero lag
- **Multi-column boards** вАФ Backlog вЖТ In Progress вЖТ Review вЖТ Done
- **Card management** вАФ create, edit, assign, prioritize, and archive tickets
- **Swimlane support** вАФ organize by team member or epic
- **Real-time column counters** вАФ always know your WIP limits

### рЯУИ Sprint Velocity Metrics
- **Live velocity charts** вАФ track story points delivered per sprint
- **Burndown visualization** вАФ monitor sprint health in real-time
- **Throughput tracking** вАФ measure team output trends over time
- **Forecast engine** вАФ project future sprint capacity from historical data
- **Custom date ranges** вАФ analyze any sprint window instantly

### рЯЧВпЄП Repository Tracking
- **Multi-repo dashboard** вАФ monitor all your repositories in one view
- **Commit activity graphs** вАФ visualize contribution patterns
- **Branch status overview** вАФ track open PRs, stale branches, and merge conflicts
- **CI/CD status indicators** вАФ see build health at a glance
- **Issue linkage** вАФ connect tickets directly to repo issues

### рЯО® Cyberpunk UI вАФ Matte Black & Electric Purple
- **Dark-first design** вАФ `#0d0d0d` base with `#7c3aed` / `#a855f7` accents
- **Glassmorphism panels** вАФ frosted-glass cards with subtle purple glows
- **Neon micro-animations** вАФ hover states, loading pulses, drag shadows
- **Responsive layout** вАФ pixel-perfect from 1080p ultrawide to 768px tablet
- **Custom scrollbars** вАФ themed to match the brand throughout

### рЯФТ Type-Safe Architecture
- **Strict TypeScript** вАФ `"strict": true` across the entire codebase
- **Full type coverage** вАФ every prop, state, API response, and event typed
- **Zero `any` tolerance** вАФ enforced via ESLint rules
- **Interface-driven design** вАФ clean contracts between all components

## рЯЪА Getting Started

### Prerequisites

Make sure you have the following installed:

| Tool | Version |
|---|---|
| [Node.js](https://nodejs.org/) | v18+ |
| [npm](https://www.npmjs.com/) | v9+ |
| [Git](https://git-scm.com/) | Latest |

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/devpulse.git

# 2. Navigate into the project
cd devpulse

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open **http://localhost:5173** вАФ the dashboard loads instantly. рЯОЙ

### Available Scripts

```bash
npm run dev        # Start Vite dev server with HMR
npm run build      # TypeScript check + production build
npm run preview    # Preview the production build locally
npm run lint       # Run ESLint across all TypeScript files
npm run type-check # Run tsc --noEmit for strict type validation
```

## рЯО® Design System вАФ Cyberpunk Theme

### Color Tokens

```css
:root {
  /* Background */
  --color-base:       #0d0d0d;   /* Matte Black вАФ main background */
  --color-surface:    #111118;   /* Elevated cards & panels */
  --color-surface-2:  #18181f;   /* Hover & active states */

  /* Electric Purple Accents */
  --color-primary:    #7c3aed;   /* Primary accent вАФ Electric Purple */
  --color-primary-lt: #a855f7;   /* Light variant вАФ glow & highlights */
  --color-primary-glow: rgba(124, 58, 237, 0.3); /* Neon glow shadow */

  /* Text */
  --color-text:       #f0ede8;   /* Primary text */
  --color-muted:      rgba(240, 237, 232, 0.45); /* Secondary text */

  /* Status */
  --color-success:    #22c55e;   /* Done / Passing */
  --color-warning:    #f59e0b;   /* At risk */
  --color-danger:     #ef4444;   /* Blocked / Failed */
  --color-info:       #38bdf8;   /* Info / In Review */
}
```
